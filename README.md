# React Movie

Applicazione web sviluppata in **React** che permette di visualizzare e cercare film utilizzando una API esterna.  
Progetto realizzato per fare pratica con React, gestione dello stato e chiamate API.

---

## Installazione e avvio

Clona la repository:

```bash
git clone https://github.com/RealComba/react-movie.git
.com/RealComba/react-movie.git 

Entra nella cartella del progetto:

cd react-movie


Installa le dipendenze:

npm install


Avvia il server di sviluppo:

npm run dev
